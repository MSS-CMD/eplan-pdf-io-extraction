## 阀岛信号命名规则

阀岛通常使用 `CXXXX_描述` 格式：
- 偶数地址（Qxxx.x0, Qxxx.x2...）= 夹紧/下降/旋转动作
- 奇数地址（Qxxx.x1, Qxxx.x3...）= 打开/伸出/复位动作

示例：Q17000.0 = C2052_夹爪气缸夹紧，Q17000.1 = C2052_夹爪气缸打开
